Prompt:
Explain the sentence in fine details, with letters used if letters are non-english. Explain one word/phrase at a time. 

SampleInput:
तेरी मेरी प्रेम कहानी, है सबसे प्यारी

Explanation:
तेरी/Teri - Yours (तेरी/Teri : yours) ; Letters used - त : Ta, र : Ra, ी : 'ee' matra
मेरी/Meri - Mine (मेरी/Meri : mine) ; Letters used - म : Ma, र : Ra, ी : 'ee' matra
प्रेम/Prem - Love (प्रेम/Prem: love); Letters used - प : Pa, र : Ra, े: 'e' matra, म: Ma
कहानी/Kahani - Story (कहानी/Kahani: story); Letters used - क: Ka, हा: Ha with 'aa' matra, न: Na, ी: 'ee' matra
है/Hai - Is (है/Hai: is); Letters used - ह: Ha, ै: 'ai' matra
सबसे/Sabse - The most/the best; Letters used - स: Sa, ब: Ba, से: Se
प्यारी/Pyari - Lovely/dear; Letters used - प्या: Pa combined with ya with 'aa' matra, री: Ra with 'ee' matra

Full Translation:
Yours and mine love story is the dearest of all.

Probable Meaning:
The speaker is expressing that the love story between them and their partner is the most precious compared to others.

Explanation:
رجعوني/Rajeouni - Return me (رجع/rajea : return, و/ou : me) ; Letters used - ر : Ra, ج : Jim, ع : Ain, و : Waw, ن : Nun, ي : Ya
عنيه/aineih - His eyes (عن/aen : about, يه/ih : him) ; Letters used - ع : Ain, ن : Nun, ي : Ya, ه : Ha
لأيام/le ayam - To days (ل/le : to, أيام/ayam : days) ; Letters used - ل : Lam, أ : Alif, ي : Ya, ا : Ain, م : Mim
اللي/elly - Which ; Letters used - ال : Alif Lam (the), ل : Lam, ي : Ya
راحوا/rahoua - They went (راح/rhah : went, وا/wa : they) ; Letters used - ر : Ra, ا : Alif, ح : Ha, و : Waw, ا : Alif
علمونا/alamouna - They taught us (علم/alim : taught, ونا/wana : us) ; Letters used - ع : Ain, ل : Lam, م : Mim, و : Waw, ن : Nun, ا : Alif
اندم /andam - Regret ; Letters used - ا : Alif, ن : Nun, د : Dal. م : Mim
على /ala - On ; Letters used - ع : Ain, ل : Lam, ى : Ya
الماضي/al madi - The Past (ال/al : the, ماضي/madi : past) ; Letters used - ال : Alif Lam (definite article), م : Mim, ا : Alif, ض : Dad, ي : Ya
وجراحوا/wejrahoua - And their wounds (و/we : and, جراح/jirah : wounds, وا/wa : their) ; Letters used - و : Waw (and), ج : Jeem, ر : Ra, ا : Alif, ح : Ha, و : Waw, ا : Alif
